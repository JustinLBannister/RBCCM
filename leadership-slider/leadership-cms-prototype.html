    Consumer GraphQL query
    
      
    
    
    
    
      
    This is the query the rbccm.com leadership component would send to WPGraphQL to get the JSON on the right:
    
      
    query GetLeadershipMembers {
  leadershipMembers(first: 100) {
    nodes {
      databaseId
      title
      leadershipFields {
        role
        imageUrl
        linkedinUrl
        bioHtml
      }
    }
  }
}
    
      
    In WordPress this is backed by a Custom Post Type leadershipMember with an ACF field group exposing role, imageUrl, linkedinUrl, and bioHtml. The WPGraphQL + WPGraphQL-for-ACF plugins do the rest.
